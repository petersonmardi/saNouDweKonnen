Android & iOS

Pafwa nou ap navige sou net la, nou pa vreman konn koman teknoloji yo fonsyone. Souvan nou jis yon senp itilizatè. Mwen pral montre nou koman teknoloji yo fonksyone e kisa ki deye yo.

Sous referans yo se an anglè e an espanyòl yo ye.

Yon ti detay sou Android e iOS , 2 sistem eksplwatasyon oubyen operating system ki plis itilize nan moman an nan zafè telefòn mobil tankou: Samsung, iPhone etsetera.

Sistem de eksplwatasyon oubyen operating system (OS) se pwogram ke apre li fin chaje otomatikman nan òdinatè a pa yon pwogram demaraj, se li ki manevre tout lòt pwogram aplikasyon ki sou yon òdinatè.

Yon Smartphone se tankou yon òdinatè tou.

Photo by Lilioni on Unsplash

Android se youn nan operating system devlope yo pa google, baze nan kernel de linux e lòt software open source oubyen kòd ke tout ka wè.

Photo by David Svihovec on Unsplash

iOS lan tou se youn nan operating system yo ki devlope pa Apple. Li de closed source (kòd ke moun pa ka wè)

Map mansyone ke tout bèl teknoloji ke nou ap wè yo, yo tout genyen kòm motè (langaj pwogramasyon oubyen programming language an anglè.

Langaj Pwogramasyon an se kisa li ye?

Pwogramasyon se kodifye,modele, simile ou prezante solisyon ak yon pwoblèm.

Li itilize yon ansanm lign ensktriksyon (kòd oubyen code an anglè) pou pèmèt yon òdinatè ekzekite sa yo mande li fè a.

Lign enstriksyon sa yo ekri nan yon langaj pwogramasyon ke moun ka konprann. Pou konprann yo fòk ou se yon pwogramè.

Langaj pwogramasyon ke nou ka jwenn se:

Java, Python, JavaScript, Swift, HTML, CSS, Objective-C, C, C Sharp, C++

Donk, Android ekri nan langaj pwogramasyon Java (entèfas itilizatè yo oubyen User Interface( UI) anglè), C (nwayo), e an C++.

iOS ekri nan langaj pwogramasyon C, C++, Objective-C, e Swift

Pou aplikasyon mobile yo, pou Android la yo ekri nan langaj pwogramasyon Java e iOS la an Swift oubyen an Objective-C.

Pa konfonn lè nou wè yon aplikasyon sou Android yon moun, epi nou menm nou ap sèvi ak iOS pou nou kouri di nou pral chèchel li. Souvan devlopè aplikasyon yo konn devlope aplikasyon pou yon sèl sistèm. A mwens ke si se ta yon gran kanpayiy, li tap devlope aplikasyon pou tout sistem yo. Konpayi tankou Facebook, Twitter, Google, Yahoo etsetera.

Pa sèlman genyen Android, iOS kòm operating system pou telefòn mobil.

Ou ap jwenn tou tankou:

Windows Phone OS, BlackBerry OS

Pou nou konn plis toujou, rete konekte ak mwen. Pi bèl enfòmasyon ap vini.
